**문제설명**

단어 s의 가운데 글자를 반환하는 함수, solution을 만들어 보세요. 단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.

**제한사항**

- s는 길이가 1 이상, 100이하인 스트링입니다.


**입출력 예**<br/>
|s|return|
|"abcde"|"c"|
|"qwer"|"we"|


<hr/>

**문제풀이**<br/>

중앙값을 계산을 먼저 구했습니다. `s`의 길이를 반으로 나눈 값을 단순히 생각할 수 있지만, 홀수일 때 x.5로 소숫점이 생깁니다. 이것을 해결하기 위해 `Math.floor` 또는 `parseInt`를 사용할 수 있습니다.
위에서 구한 index를 `mid`라고 했을 때 `s.length`가 2로 나누어 떨어진다면(짝수라면) `s[mid - 1] + s[mid]`를 그렇지 않다면 `s[mid]`를 반환하였습니다.
